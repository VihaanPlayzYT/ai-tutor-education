# AI Tutor for Underprivileged Students

## Project Overview
A machine learning project that aims to make quality education more accessible. This AI Tutor helps classify student questions and suggests learning resources.

**Why this project?**  
As the IT Head at my school and a volunteer teacher for underprivileged students, I saw first-hand the challenges they face. This project uses AI to provide instant support and personalized guidance.

## Features (Planned)
- Question classification (Math / Science / English)
- Simple recommendation system for learning resources
- User-friendly interface (to be added later)

## Technologies Used
- Python
- scikit-learn (for ML models)
- Pandas & NumPy
- Google Colab (for development)

## How It Works
1. Student inputs a question.
2. Model classifies the subject/topic.
3. System suggests relevant study material or explanation.
